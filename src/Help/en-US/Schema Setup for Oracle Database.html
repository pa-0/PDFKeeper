Schema Setup for Oracle Database

This topic provides the steps that need to be performed by the individual responsible for the administration of the Oracle Database to setup the schema for PDFKeeper.

Note, if Oracle Database was installed on a different computer, you will need to download Oracle Instant Client Basic or Basic Lite and SQL*Plus packages to this computer, extract the ZIP files into an empty folder, and then add that folder to the PATH. As an alternative, you can copy OracleDatabaseSchemaSetup.cmd and OracleDatabaseSchemaSetup.sql from %LOCALAPPDATA%\Programs\PDFKeeper to the computer hosting Oracle Database and execute there, just substitute the folder path in step 2 with the location where the files were copied to.

  1. From Windows, right click on the Start button and then select Run.

  2. Paste %LOCALAPPDATA%\Programs\PDFKeeper\OracleDatabaseSchemaSetup.cmd into the combo box and then select OK.

  3. Select option 1 and then follow the instructions.
