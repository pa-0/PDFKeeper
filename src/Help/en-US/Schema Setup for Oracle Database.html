Schema Setup for Oracle Database

This topic provides the steps that need to be performed by the individual responsible for the administration of the Oracle Database to setup the schema for PDFKeeper.

  1. If Oracle Database was installed on a different computer, you will need to download Oracle Instant Client Basic or Basic Lite and SQL*Plus packages to this computer, extract the ZIP files into an empty folder, and then add that folder to the PATH. As an alternative, you can install PDFKeeper on the same computer as the Oracle Database and perform the setup on that computer.

  2. From Windows, right click on the Start button and then select Run.

  3. Paste %LOCALAPPDATA%\Programs\PDFKeeper\MultiUserDatabaseSchemaSetup.cmd into the combo box and then select OK.

  4. Select option 1 and then follow the instructions.
